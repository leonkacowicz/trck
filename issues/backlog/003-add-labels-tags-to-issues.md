# Add labels/tags to issues

## Summary
Add a `labels` array field to issues, with verbs to add/remove labels and filter by them. Builds on preserve-unknown-keys for forward-compat.

## Acceptance criteria
- [ ] `trck label NNN --add X --remove Y`
- [ ] `trck list --label X`
- [ ] labels shown in `show`/`summary`
