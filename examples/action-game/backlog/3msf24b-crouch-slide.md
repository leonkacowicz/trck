# Crouch & slide

## Summary
Crouch (shrinks the hurtbox, slows movement) and a momentum-based slide that carries run
speed under low gaps.

## Acceptance criteria
- [ ] Crouching halves the hurtbox height
- [ ] Slide preserves run momentum and ends in a crouch
- [ ] Cannot slide from standstill
