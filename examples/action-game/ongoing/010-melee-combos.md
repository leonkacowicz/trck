# Melee combos

## Summary
Three-hit melee combo with a short cancel window into dash. Each hit has its own
startup/active/recovery timing and a forgiving input buffer.

## Acceptance criteria
- [ ] Three-hit ground combo chains within the buffer window
- [ ] Final hit has knockback
- [ ] Combo can be dash-cancelled after the active frames
